# 第3章 统计多径信道模型

## 3.1 引言：为什么信号会快速波动？

在上一章讨论的大尺度衰落（路径损耗和阴影衰落）基础上，本章聚焦**小尺度衰落**——信号在**波长量级的距离**或**毫秒量级的时间**内发生的快速随机波动。这种波动源于**多径传播**：发射信号经不同路径（反射、衍射、散射）到达接收机，各路径信号因传播距离不同而具有不同的**时延**和**相位**，在接收端叠加时可能**相长**或**相消**，导致接收信号强度剧烈变化。

理解小尺度衰落对设计可靠的无线链路至关重要，它直接影响调制方式、分集方案和信道编码的选择。

## 3.2 多径信道的基本数学模型

### 3.2.1 时变冲激响应
多径信道可建模为一个**线性时变系统**，用**时变冲激响应** \(c(\tau, t)\) 描述：
- \(\tau\)：时延变量，表示多径分量的到达时间
- \(t\)：时间变量，表示信道的时变性（由发射机、接收机或散射体运动引起）

接收信号 \(r(t)\) 是发送信号 \(s(t)\) 与信道冲激响应的卷积：
\[
r(t) = \int_{-\infty}^{\infty} c(\tau, t) s(t-\tau) d\tau
\]

### 3.2.2 多径分量的典型表达
若信道有 \(L\) 条可分辨路径，每条路径具有复增益 \(a_l(t)\) 和时延 \(\tau_l(t)\)，则：
\[
c(\tau, t) = \sum_{l=1}^{L} a_l(t) e^{-j\phi_l(t)} \delta(\tau - \tau_l(t))
\]
其中相位 \(\phi_l(t) = 2\pi f_c \tau_l(t)\)，\(f_c\) 为载波频率。

## 3.3 多径信道的关键特征参数

### 3.3.1 时延扩展与相干带宽
- **时延扩展**：多径分量到达时间的展宽程度，是信道在**时域**的色散度量。
  - **平均超额时延**：\(\overline{\tau} = \frac{\sum_k P_k \tau_k}{\sum_k P_k}\)
  - **均方根时延扩展**：\(\sigma_\tau = \sqrt{\overline{\tau^2} - \overline{\tau}^2}\)，其中 \(\overline{\tau^2} = \frac{\sum_k P_k \tau_k^2}{\sum_k P_k}\)
- **相干带宽** \(B_c\)：信道频率响应保持强相关（相关系数>0.5）的频率范围。近似有：
  \[
  B_c \approx \frac{1}{5\sigma_\tau}
  \]

**重要推论**：
- 若信号带宽 \(B_s \ll B_c\)，则所有频率分量经历相似衰落 → **平坦衰落**
- 若 \(B_s > B_c\)，则不同频率分量衰落不同 → **频率选择性衰落**，导致**码间干扰(ISI)**

### 3.3.2 多普勒扩展与相干时间
- **多普勒频移**：由相对运动引起，\(f_d = \frac{v}{c}f_c \cos\theta\)，其中 \(v\) 为相对速度，\(\theta\) 为运动方向与波达方向的夹角。
- **多普勒扩展** \(B_d\)：多普勒频移的展宽范围，反映信道在**频域**的时变速度。
- **相干时间** \(T_c\)：信道冲激响应保持强相关的时间长度。近似有：
  \[
  T_c \approx \frac{1}{B_d}
  \]

**重要推论**：
- 若符号周期 \(T_s \ll T_c\)，则信道在一个符号期间变化缓慢 → **慢衰落**
- 若 \(T_s > T_c\)，则信道在一个符号期间快速变化 → **快衰落**

### 3.3.3 总结：信道分类
| 信道类型 | 条件 | 主要影响 |
|----------|------|----------|
| **平坦慢衰落** | \(B_s \ll B_c\)，\(T_s \ll T_c\) | 信号整体均匀衰减，且衰减基本不变 |
| **频率选择性慢衰落** | \(B_s > B_c\)，\(T_s \ll T_c\) | 引起码间干扰(ISI)，但信道变化慢 |
| **平坦快衰落** | \(B_s \ll B_c\)，\(T_s > T_c\) | 信号整体均匀衰减，但衰减快速变化 |
| **频率选择性快衰落** | \(B_s > B_c\)，\(T_s > T_c\) | 既有ISI，又快速变化，最难处理 |

实际系统中，**频率选择性慢衰落**和**平坦慢衰落**更常见。快衰落通常发生在高速移动场景。

## 3.4 小尺度衰落的统计模型

### 3.4.1 瑞利衰落
当接收信号由大量**独立同分布**的散射分量叠加而成，且**不存在主导的直射路径**时，接收信号的复包络 \(r = r_I + j r_Q\) 的实部 \(r_I\) 和虚部 \(r_Q\) 为独立零均值高斯过程。此时信号包络 \(R = |r|\) 服从**瑞利分布**：

\[
f_R(r) = \frac{r}{\sigma^2} \exp\left(-\frac{r^2}{2\sigma^2}\right), \quad r \geq 0
\]

其中 \(2\sigma^2 = E[R^2]\) 是平均包络功率。

**重要特性**：
- 相位均匀分布
- 平均信噪比(SNR)为 \(\overline{\gamma} = \frac{E_b}{N_0} E[R^2]\)
- 瞬时SNR \(\gamma\) 服从指数分布：\(f_\gamma(\gamma) = \frac{1}{\overline{\gamma}} e^{-\gamma/\overline{\gamma}}\)

瑞利衰落是**恶劣衰落环境**的典型模型，如城市环境中无直射路径的移动通信。

### 3.4.2 莱斯衰落
当存在一条稳定的**主导路径**（通常是直射路径）时，接收信号复包络为高斯过程加上一个**非零均值**。此时包络 \(R\) 服从**莱斯分布**：

\[
f_R(r) = \frac{r}{\sigma^2} \exp\left(-\frac{r^2 + s^2}{2\sigma^2}\right) I_0\left(\frac{rs}{\sigma^2}\right), \quad r \geq 0
\]

其中：
- \(s^2\) 是主导分量的功率
- \(I_0(\cdot)\) 是零阶修正贝塞尔函数
- **莱斯因子** \(K = \frac{s^2}{2\sigma^2}\) 表示主导分量功率与散射分量功率之比

**特殊情形**：
- 当 \(K=0\)（无主导路径）时，莱斯分布退化为瑞利分布
- 当 \(K \to \infty\) 时，退化为高斯分布（无衰落）

莱斯衰落是**较好传播条件**的模型，如郊区、农村或有直射路径的室内环境。

### 3.4.3 Nakagami-m衰落
Nakagami-m分布是更通用的经验模型，其包络分布为：

\[
f_R(r) = \frac{2m^m r^{2m-1}}{\Gamma(m)\Omega^m} \exp\left(-\frac{m}{\Omega}r^2\right), \quad r \geq 0
\]

其中：
- \(\Omega = E[R^2]\) 是平均接收功率
- \(m \geq 1/2\) 是衰落系数（形状参数）
- \(\Gamma(\cdot)\) 是Gamma函数

**与其他分布的关系**：
- \(m=1\)：退化为瑞利分布
- \(m=(K+1)^2/(2K+1)\) 时，近似莱斯分布
- \(m \to \infty\)：无衰落（确定性信道）

Nakagami-m分布因灵活性高而在实测数据拟合中常用。

## 3.5 衰落信道的动态特性

### 3.5.1 电平通过率
**电平通过率(LCR)** 定义为信号包络在单位时间内以正斜率穿过某一特定电平 \(R\) 的平均次数。对于瑞利衰落：

\[
\text{LCR}_R = \sqrt{2\pi} f_d \rho e^{-\rho^2}
\]

其中：
- \(f_d\) 是最大多普勒频移
- \(\rho = R/R_{\text{rms}}\) 是归一化电平（\(R_{\text{rms}} = \sqrt{E[R^2]}\)）

LCR反映了衰落的**快速程度**。高LCR意味着信号频繁跌至低电平，对系统设计提出更高要求。

### 3.5.2 平均衰落持续时间
**平均衰落持续时间(AFD)** 定义为信号包络低于特定电平 \(R\) 的平均时间长度。对于瑞利衰落：

\[
\text{AFD}_R = \frac{e^{\rho^2} - 1}{\rho f_d \sqrt{2\pi}}
\]

AFD与LCR的关系：\(\text{AFD}_R = \frac{P(R_{\text{envelope}} < R)}{\text{LCR}_R}\)

**系统设计意义**：
- AFD决定了信道处于深衰落的典型时长
- 影响交织器深度、重传机制等设计
- 例如，若AFD为10ms，则交织器应跨越多于10ms以打散突发错误

## 3.6 多径信道的系统理论描述

### 3.6.1 广义平稳非相关散射模型
为简化分析，常采用**广义平稳非相关散射(WSSUS)** 假设：
- **广义平稳(WSS)**：信道统计特性在短时间内不变
- **非相关散射(US)**：不同时延的多径分量互不相关

在此假设下，信道相关函数可分解为：
- **时延自相关函数**（多径强度剖面）：描述不同时延分量的平均功率
- **时间自相关函数**：反映信道时变特性，傅里叶变换给出**多普勒功率谱**

### 3.6.2 散射函数
**散射函数** \(S(\tau, f)\) 是同时描述时延扩展和多普勒扩展的二维函数：
- 对时延 \(\tau\) 的积分给出多普勒功率谱
- 对多普勒频率 \(f\) 的积分给出功率时延谱

它是设计和分析接收机（如均衡器、匹配滤波器）的重要工具。

## 3.7 离散时间信道模型

为便于数字仿真和接收机设计，需建立**离散时间信道模型**。将连续信道冲激响应 \(c(\tau, t)\) 在时延域以间隔 \(T_s\)（符号周期）采样，得到**抽头延迟线模型**：

\[
h[n] = \sum_{l=0}^{L-1} h_l[n] \delta[n-l]
\]

其中 \(h_l[n]\) 是第 \(l\) 个抽头在时刻 \(n\) 的复增益，通常建模为复高斯随机过程。

**模型参数确定**：
- 抽头数 \(L = \lceil \tau_{\max}/T_s \rceil + 1\)，\(\tau_{\max}\) 为最大时延扩展
- 各抽头功率由功率时延谱确定
- 各抽头的时变特性由多普勒谱确定

## 3.8 空时信道模型（MIMO信道）

对于多天线系统，信道扩展为矩阵形式。\(N_t\) 根发射天线和 \(N_r\) 根接收天线之间的信道可用矩阵 \(\mathbf{H} \in \mathbb{C}^{N_r \times N_t}\) 描述，其中元素 \(h_{ij}\) 是发射天线 \(j\) 到接收天线 \(i\) 的复增益。

### 3.8.1 关键特性
- **空间相关性**：天线间距不足导致信道系数相关，降低MIMO性能
- **键孔效应**：即使天线不相关，信道秩也可能很低（如丰富散射环境中的关键孔传播）

### 3.8.2 常用模型
- **独立同分布模型**：\(h_{ij}\) 为独立复高斯随机变量，适用于天线间距足够、丰富散射环境
- **相关模型**：\(\mathbf{H} = \mathbf{R}_r^{1/2} \mathbf{H}_w \mathbf{R}_t^{1/2}\)，其中 \(\mathbf{R}_r, \mathbf{R}_t\) 为接收/发送相关矩阵，\(\mathbf{H}_w\) 为独立同分布矩阵
- **几何模型**：基于散射体几何分布推导，如克拉美-劳模型

## 3.9 信道模型的实测与验证

### 3.9.1 测量方法
- **时域方法**：发射脉冲序列，直接测量冲激响应
- **频域方法**：发射频率扫描信号，测量频率响应后反变换
- **扩频方法**：使用伪随机序列，利用其良好自相关特性

### 3.9.2 参数提取
从测量数据中提取：
- 功率时延谱
- 时延扩展
- 多普勒谱
- 空间相关性等

### 3.9.3 标准化信道模型
为便于系统评估和比较，标准化组织定义了典型信道模型：
- **ITU模型**：如车载、行人、室内等场景
- **3GPP模型**：如EPA、EVA、ETU等LTE信道模型
- **IEEE 802.11模型**：室内WLAN信道模型

## 3.10 总结

本章系统阐述了统计多径信道模型的核心内容：

1. **多径效应本质**：信号经多条路径传播，产生时延扩展和多普勒扩展，导致小尺度衰落。

2. **关键参数**：
   - **时延扩展** → 相干带宽 → 频率选择性/平坦衰落
   - **多普勒扩展** → 相干时间 → 快/慢衰落

3. **统计模型**：
   - **瑞利衰落**：无主导路径，恶劣环境
   - **莱斯衰落**：存在直射路径，较好环境
   - **Nakagami-m衰落**：灵活的经验模型

4. **动态特性**：电平通过率和平均衰落持续时间量化了衰落的快速性和严重性。

5. **系统建模**：离散抽头延迟线模型便于仿真和接收机设计。

6. **扩展**：MIMO信道考虑空间维度，空时信道模型更为复杂。

理解小尺度衰落是设计抗衰落技术（如分集、均衡、信道编码）的基础。在后续章节中，将看到如何利用这些知识设计可靠的无线通信系统。衰落虽是挑战，但通过合理设计，其不利影响可被显著抑制，甚至可被利用（如MIMO系统的空间分集和复用）。
