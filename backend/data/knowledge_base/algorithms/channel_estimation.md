# 信道估计

## 概述

信道估计是无线接收机的核心环节。接收机通过已知导频信号估计信道响应，用于后续的均衡和检测。估计精度直接影响系统性能。

## 导频结构

### 块状导频 (Block-type Pilot)
- 在特定 OFDM 符号上所有子载波都是导频
- 适用于慢衰落信道
- 典型：IEEE 802.11a 长训练序列

### 梳状导频 (Comb-type Pilot)
- 在特定子载波上持续传输导频
- 适用于快衰落信道
- 需插值恢复完整信道响应

### 格状导频 (Lattice-type Pilot)
- 时频二维分散放置
- 5G NR DM-RS 采用此方案
- 兼顾时频选择性

## 经典算法

### LS (最小二乘) 估计

```
H_LS = Y_p / X_p = H + N / X_p
```

- 计算简单，无需信道先验信息
- 缺点：对噪声敏感，没有利用信道相关性

### MMSE (最小均方误差) 估计

```
H_MMSE = R_HH * (R_HH + σ² * (X * X^H)^{-1})^{-1} * H_LS
```

- 利用信道二阶统计量，抑制噪声
- 估计精度优于 LS，约 3~5 dB 增益
- 缺点：需知道信道相关矩阵和噪声方差

### 压缩感知估计

- 利用无线信道在时延-多普勒域的稀疏性
- OMP、CoSaMP 等算法可大幅减少导频开销
- 适用于 Massive MIMO 系统

## 插值方法

对于梳状导频，需在频域插值得到完整信道：
- **线性插值**：简单，精度一般
- **DFT 插值**：利用时域加窗抑制泄漏
- **维纳插值**：最优 MMSE 插值器，复杂度高

## 性能评估

常用指标：
- **MSE** (均方误差)：信道估计值与真实值的偏差
- **BER / SER** (误码率/误符号率)：估计质量对检测性能的影响
- **导频开销**：导频占用的时频资源比例
