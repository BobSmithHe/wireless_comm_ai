# OFDM (正交频分复用)

## 概述

OFDM 是现代无线通信（4G LTE、5G NR、WiFi 6/7）的核心物理层技术。它将高速数据流分配到多个正交子载波上并行传输，将频率选择性衰落信道转化为多个平坦衰落子信道。

## 核心原理

1. **串并转换**：将高速数据流分解为 N 路低速并行子数据流
2. **子载波调制**：每路子数据流调制到正交子载波上（QPSK / 16QAM / 64QAM）
3. **IFFT 实现**：利用 IFFT 高效生成正交子载波，子载波间隔 Δf = B / N
4. **循环前缀 (CP)**：在每个 OFDM 符号前添加 CP，长度大于最大多径时延，消除 ISI
5. **FFT 解调**：接收端去除 CP 后做 FFT，恢复各子载波数据

## 关键参数

| 参数 | LTE (20MHz) | 5G NR (FR1 100MHz) |
|------|-------------|---------------------|
| 子载波间隔 | 15 kHz | 30 kHz |
| FFT 点数 | 2048 | 4096 |
| CP 长度 | 144 / 160 | 320 |
| 符号周期 | 66.7 μs | 33.3 μs |

## 优点与挑战

**优点**：
- 频谱效率高（子载波正交重叠）
- 抗多径衰落能力强（CP + 窄带子载波）
- 实现简单（FFT/IFFT）

**挑战**：
- 对频偏敏感（破坏正交性 → ICI）
- PAPR 高（多载波叠加 → 功放效率低）
- 对时间同步要求严格

## 应用场景

- 4G LTE / 5G NR 下行
- WiFi (802.11a/g/n/ac/ax/be)
- DVB-T / DAB 数字广播
- ADSL / VDSL 有线接入
