# 第2章 路径损耗和阴影衰落

## 2.1 引言：无线信号如何衰减？

当无线信号在空间传播时，其功率会随距离增加而衰减。这种衰减主要来自两种物理机制：

1.  **路径损耗**：由波前扩展和传播环境引起的确定性衰减，是距离的函数。
2.  **阴影衰落**：由大型障碍物（如山丘、建筑物）遮挡引起的随机衰减，服从统计规律。

理解这两种机制是设计可靠无线系统的第一步。本章将建立数学模型来描述它们，并分析其对网络覆盖和性能的影响。

## 2.2 无线电波传播机制

信号从发射端到接收端可能通过多种路径，每种路径对应不同的物理机制：

- **直射**：视距路径，信号无遮挡直接传播。
- **反射**：信号遇到比波长大得多的平滑表面（如地面、墙壁）发生反射。
- **衍射**：信号遇到障碍物边缘时发生弯曲绕射，可传播到阴影区域。
- **散射**：信号遇到粗糙表面或尺寸与波长相当的物体时向各个方向散射。

在实际环境中，接收信号通常是这些多径分量叠加的结果，但路径损耗模型主要关注平均功率的衰减。

## 2.3 自由空间路径损耗：最简单的起点

自由空间是一种理想环境，无反射、衍射和散射，只有波前扩展导致的衰减。

**Friis自由空间传输公式**给出了发射功率 \(P_t\) 与接收功率 \(P_r\) 的关系：

\[
P_r = P_t G_t G_r \left( \frac{\lambda}{4\pi d} \right)^2
\]

其中：
- \(G_t, G_r\)：发射和接收天线增益
- \(\lambda\)：波长
- \(d\)：发射机与接收机之间的距离

**路径损耗(PL)**定义为发射功率与接收功率的比值（通常用dB表示）：

\[
PL_{\text{dB}} = 10\log_{10}\left(\frac{P_t}{P_r}\right) = 20\log_{10}\left(\frac{4\pi d}{\lambda}\right) - G_{t,\text{dB}} - G_{r,\text{dB}}
\]

**关键观察**：
- 自由空间路径损耗与 \(d^2\) 成正比（即功率与 \(1/d^2\) 成正比）。
- 与频率平方成正比（因为 \(\lambda = c/f\)，所以 \(PL \propto f^2\)）。

自由空间模型是理论基准，但实际环境复杂得多。

## 2.4 实际环境中的路径损耗：经验模型

在实际环境中，路径损耗不仅来自波前扩展，还与地形、建筑物、植被等密切相关。通过大量测量数据，人们总结出**经验模型**。最经典的模型之一是**对数距离路径损耗模型**：

\[
PL(d)_{\text{dB}} = PL(d_0)_{\text{dB}} + 10n \log_{10}\left(\frac{d}{d_0}\right) + X_\sigma
\]

其中：
- \(d_0\)：参考距离（通常取1m或1km）
- \(PL(d_0)\)：参考距离处的路径损耗（由自由空间公式计算或测量得到）
- \(n\)：**路径损耗指数**，是模型的关键参数
- \(X_\sigma\)：**阴影衰落**项，是一个均值为0、标准差为\(\sigma\)（单位dB）的高斯随机变量

**路径损耗指数 \(n\) 的典型值**：
- 自由空间：\(n = 2\)
- 市区蜂窝环境：\(n = 2.7 \sim 3.5\)
- 有障碍物的室内：\(n = 4 \sim 6\)
- 有严重遮挡的工厂：\(n = 2 \sim 3\)

**阴影衰落 \(X_\sigma\)** 反映了在相同距离下，由于环境差异（如建筑物布局不同）导致的接收功率波动。其标准差\(\sigma\)通常为4-12 dB。

## 2.5 几种著名的经验模型

### 2.5.1 Okumura-Hata模型
基于东京地区大量测量数据，频率范围150-1500MHz，距离1-20km。Hata将Okumura的曲线拟合为公式，便于计算。

**市区路径损耗公式**（单位dB）：

\[
L_{\text{urban}} = 69.55 + 26.16\log_{10}(f) - 13.82\log_{10}(h_b) - a(h_m) + [44.9 - 6.55\log_{10}(h_b)]\log_{10}(d)
\]

其中：
- \(f\)：频率(MHz)
- \(h_b\)：基站天线有效高度(m)
- \(h_m\)：移动台天线高度(m)
- \(a(h_m)\)：移动台天线高度修正因子
- \(d\)：距离(km)

该模型是宏蜂窝系统规划的重要工具。

### 2.5.2 COST-231 Hata模型
Okumura-Hata模型的扩展，将频率扩展到2GHz，适用于中小城市。

### 2.5.3 室内传播模型
室内环境更为复杂，有**对数距离模型**、**衰减因子模型**等。**ITU室内路径损耗模型**简单实用：

\[
L = 20\log_{10}(f) + n\log_{10}(d) + L_f(n_f) - 28 \quad (\text{dB})
\]

其中：
- \(f\)：频率(MHz)
- \(n\)：路径损耗指数（住宅/办公室：2.8；商业楼：3.3）
- \(d\)：距离(m)
- \(L_f\)：楼层穿透损耗因子(dB)
- \(n_f\)：穿透的楼层数

## 2.6 阴影衰落：信号的大尺度波动

即使距离固定，由于大型障碍物的遮挡，接收信号功率也会在平均功率上下波动。这种波动称为阴影衰落，通常建模为**对数正态分布**。

### 2.6.1 对数正态阴影衰落
若用dB表示接收功率，阴影衰落服从**正态分布**（高斯分布）：

\[
X_\sigma \sim \mathcal{N}(0, \sigma^2)
\]

其中\(\sigma\)为标准差，典型值：
- 市区宏蜂窝：6-12 dB
- 郊区：4-8 dB
- 室内：4-12 dB（取决于墙壁材料、布局）

若用线性标度表示接收功率，则服从**对数正态分布**。

### 2.6.2 阴影衰落的相关性
阴影衰落具有空间相关性：相邻位置的阴影衰落值相似。相关性随距离增加而减小，通常用**相关距离**描述（典型值：几十米到几百米）。相关距离内，阴影衰落值变化不大。

## 2.7 路径损耗与阴影衰落的结合模型

结合路径损耗和阴影衰落，接收功率可表示为：

\[
P_r(d)_{\text{dBm}} = P_t_{\text{dBm}} + G_t_{\text{dBi}} + G_r_{\text{dBi}} - PL(d_0)_{\text{dB}} - 10n\log_{10}\left(\frac{d}{d_0}\right) - X_\sigma
\]

这是一个**大尺度衰落模型**，用于分析网络覆盖和系统性能。

## 2.8 对系统设计的影响

### 2.8.1 中断概率
由于阴影衰落，即使平均接收功率足够，瞬时功率也可能低于接收机灵敏度，导致通信中断。

中断概率定义为：

\[
P_{\text{out}} = P[P_r(d) < P_{\min}]
\]

对于对数正态阴影衰落，可推导出：

\[
P_{\text{out}} = Q\left( \frac{P_{\min} - \overline{P_r}(d)}{\sigma} \right)
\]

其中：
- \(Q(x) = \frac{1}{\sqrt{2\pi}}\int_x^\infty e^{-t^2/2} dt\) 是Q函数
- \(\overline{P_r}(d)\) 是平均接收功率（仅考虑路径损耗）
- \(P_{\min}\) 是接收机灵敏度

**设计启示**：为降低中断概率，需要增加**链路预算余量**（即提高发射功率或降低接收机灵敏度），或通过**分集**技术对抗阴影衰落。

### 2.8.2 小区覆盖范围
考虑一个半径为\(R\)的圆形小区。由于阴影衰落，小区内某些点可能接收信号弱。我们关心**小区覆盖面积比例**（即接收功率大于\(P_{\min}\)的区域比例）。

经推导，覆盖面积比例\(F_u\)为：

\[
F_u = \frac{1}{2} \left[ 1 - \text{erf}(a) + \exp\left( \frac{1-2ab}{b^2} \right) \left( 1 - \text{erf}\left( \frac{1-ab}{b} \right) \right) \right]
\]

其中：
- \(a = \frac{P_{\min} - \overline{P_r}(R)}{\sqrt{2}\sigma}\)
- \(b = \frac{10n\log_{10}(e)}{\sqrt{2}\sigma}\)
- \(\text{erf}(x) = \frac{2}{\sqrt{\pi}} \int_0^x e^{-t^2} dt\)

**设计启示**：
- 当\(n\)和\(\sigma\)给定时，可通过调整发射功率来控制覆盖面积比例。
- 覆盖面积比例通常达不到100%，需通过重叠覆盖（蜂窝结构）来保证连续性。

### 2.8.3 频率选择与规划
- 低频信号（如700MHz）路径损耗小，穿透能力强，适合广覆盖。
- 高频信号（如3.5GHz、毫米波）路径损耗大，但带宽大，适合热点容量补充。
- 网络规划时需根据频段特性选择合适的基站间距和功率。

## 2.9 实际考虑与测量

### 2.9.1 模型校准
经验模型给出典型值，但实际环境千差万别。网络部署前需进行**路测**，测量接收功率随距离的变化，从而校准本地化的路径损耗指数\(n\)和阴影标准差\(\sigma\)。

### 2.9.2 链路预算
链路预算是计算最大允许路径损耗的系统设计工具。基本公式：

\[
\text{最大路径损耗} = \text{发射功率} + \text{发射天线增益} + \text{接收天线增益} - \text{接收机灵敏度} - \text{余量}
\]

其中余量包括：
- 阴影衰落余量：根据所需覆盖概率（如90%）和\(\sigma\)确定
- 干扰余量：考虑系统内干扰
- 穿透损耗余量：考虑车辆、建筑穿透损耗
- 人体损耗：手机靠近人体时的额外损耗

## 2.10 总结

本章介绍了无线信道大尺度衰落的两个核心组成部分：

1.  **路径损耗**：确定性衰减，与距离的\(n\)次方成反比，是网络覆盖范围的主要决定因素。
2.  **阴影衰落**：随机性衰减，服从对数正态分布，是影响覆盖可靠性的关键。

两者结合决定了接收信号的平均功率及其波动范围。通过建立数学模型，我们可以：
- 预测特定距离下的平均接收功率
- 计算通信中断概率
- 设计满足覆盖要求的小区半径
- 进行合理的频率规划和功率分配

这些分析构成了无线网络规划与优化的基础。后续章节将讨论**小尺度衰落**（多径效应），它导致信号在波长量级距离上快速波动，是影响信号质量与可靠性的另一关键因素。路径损耗、阴影衰落和小尺度衰落共同构成了完整的无线信道衰减图景。
