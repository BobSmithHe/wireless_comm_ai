# 第1章 无线通信概述

## 1.1 无线通信的历史

无线通信的历史是一部从基础科学发现到全球互联的演进史。其核心奠基人是19世纪的几位科学巨匠。

**电磁理论的奠基（19世纪）**：
- **迈克尔·法拉第**（1831年）：发现电磁感应现象，证明变化的磁场能产生电场。
- **詹姆斯·克拉克·麦克斯韦**（1864年）：用一组方程统一了电与磁，预言了电磁波的存在。
- **海因里希·赫兹**（1887年）：通过实验首次产生并检测到无线电波，证实了麦克斯韦的理论。频率单位“赫兹(Hz)”以他命名。

**从实验到应用（19世纪末-20世纪初）**：
关键转折点出现在**伽利尔摩·马可尼**的工作。1895年，他成功实现了无线电报通信。1901年，他完成了划时代的**跨大西洋无线电信号传输**，证明了远距离无线通信的可行性，并因此获得1909年诺贝尔物理学奖。这一时期，无线通信主要用于船舶间的电报通信。

**广播与大众化（20世纪早期-中期）**：
- **广播时代开启**：1906年，雷金纳德·费森登进行了首次语音广播。1920年代，AM广播电台（如KDKA）开始定期播音，无线电成为首个大众电子媒体。
- **移动通信萌芽**：早期的移动电话系统笨重且昂贵，如1946年的MTS，但奠定了蜂窝概念的基础。

**蜂窝革命与数字化（20世纪后期）**：
- **第一代(1G, 1980s)**：模拟蜂窝系统（如AMPS），仅提供语音服务。
- **第二代(2G, 1990s)**：数字系统（如GSM），引入TDMA/CDMA，支持短信和低速数据。
- **从2.5G到3G**：GPRS/EDGE带来移动互联网，WCDMA等3G标准支持视频通话和更高速数据。

**宽带与融合时代（21世纪至今）**：
- **4G/LTE**（2010s）：基于OFDMA和全IP网络，提供真正的移动宽带体验。
- **5G**（2019+）：不仅追求更高速度，更支持**大规模机器通信(mMTC)** 和**超可靠低时延通信(URLLC)**，赋能物联网和垂直行业。
- **无处不在的连接**：Wi-Fi、蓝牙、ZigBee等短距技术与广域蜂窝网络互补，构成了立体的无线连接世界。

## 1.2 无线愿景

未来无线通信的愿景是构建一个**智能、泛在、融合**的数字世界。其核心驱动力来自三大类应用场景，由国际电信联盟(ITU)为5G定义：

1.  **增强型移动宽带(eMBB)**：追求**极致的速度与容量**，支持8K/AR/VR、全息通信等体验。
2.  **超可靠低时延通信(URLLC)**：追求**极致的可靠性与实时性**（时延<1ms，可靠性>99.999%），是工业自动化、远程手术、自动驾驶的基石。
3.  **海量机器类通信(mMTC)**：追求**极致的连接密度**（每平方公里百万连接），支撑智慧城市、环境监测等物联网爆发。

面向未来的**6G愿景**（预计2030年商用）更为宏大，可能包含：
- **太赫兹通信**：使用0.1-10THz频段，开启太比特每秒(Tbps)速率时代。
- **通信感知一体化**：网络不仅能通信，还能像雷达一样感知环境。
- **人工智能原生**：AI深度融入网络设计、运维与优化。
- **全域覆盖**：空、天、地、海一体化无缝网络。

## 1.3 技术问题

实现上述愿景面临着一系列严峻的技术挑战，这些挑战也构成了本书后续章节要解决的核心问题：

| 挑战类别 | 具体问题 | 影响与解决思路 |
| :--- | :--- | :--- |
| **频谱稀缺** | 黄金频段（<6GHz）已近乎饱和。 | 推动**频谱共享**（如动态频谱接入、认知无线电），并向**高频段（毫米波、太赫兹）** 拓展。 |
| **信道损伤** | 信号在无线信道中经历**路径损耗、阴影衰落和多径衰落**，导致衰减、失真和不稳定。 | 采用**信道编码、均衡、分集**等技术对抗。 |
| **多径衰落** | 信号经不同路径到达，产生**时延扩展**（导致码间干扰）和**多普勒扩展**（导致频率偏移）。 | 利用**OFDM**对抗频率选择性衰落，利用**信道估计与均衡**对抗时延扩展。 |
| **干扰** | 包括**同频干扰(CCI)**、**邻频干扰(ACI)** 和**用户间干扰**。 | 通过**功率控制、智能天线、先进的多址接入（如NOMA）** 来管理。 |
| **移动性** | 用户移动导致**多普勒频移**和频繁的**越区切换**。 | 设计高效的切换算法和鲁棒的信道估计。 |
| **能耗** | 海量设备和基站带来巨大能耗。 | 发展**高能效硬件、智能休眠机制、无线携能通信**。 |
| **安全与隐私** | 无线广播的开放性带来安全隐患。 | 结合**高强度加密**与**物理层安全**技术。 |

## 1.4 现有无线系统

今天的无线世界是一个由多种技术构成的生态系统，各有其定位。

1.  **蜂窝网络**：提供广域、连续覆盖的移动通信。
    - **从1G到5G的演进**：1G（模拟语音）→ 2G（数字语音/短信）→ 3G（移动数据）→ 4G（移动宽带）→ 5G（万物互联）。每一代都在速率、容量和连接能力上实现飞跃。

2.  **无线局域网(WLAN, Wi-Fi)**：提供局部区域的高速接入。
    - 基于IEEE 802.11系列标准，工作在2.4GHz和5GHz非授权频段。
    - 从802.11a/b/g（54Mbps）发展到802.11ax（Wi-Fi 6，理论速率近10Gbps），引入了OFDMA、上行MU-MIMO等关键技术。

3.  **个域网(PAN)**：实现设备间短距离互联。
    - **蓝牙**：主打低功耗，连接耳机、鼠标、音箱等。
    - **ZigBee**：专为低功耗、低数据速率的物联网设备设计。
    - **超宽带(UWB)**：提供极高精度（厘米级）的室内定位。

4.  **卫星通信**：实现无地域限制的全球覆盖。
    - 轨道类型：**地球同步轨道(GEO)**、**中地球轨道(MEO)**、**低地球轨道(LEO)**。
    - 新一代LEO星座（如**星链Starlink**）旨在提供全球宽带互联网。

5.  **广播系统**：实现一点对多点的信息分发。
    - **数字音频广播(DAB)** 和**数字视频广播(DVB)** 提供了比模拟广播更优质、高效的音视频服务。

## 1.5 无线频谱

频谱是无线通信的**核心稀缺资源**，由国家与国际机构严格管理。

- **管理机构**：国际电信联盟(ITU)负责全球协调，各国机构（如美国FCC、中国工信部）负责本国分配。

- **关键频段特性**：
  - **低频(<1GHz)**：传播损耗小，覆盖广，但带宽窄。用于**广覆盖**（如5G的700MHz）。
  - **中频(1-6GHz)**：覆盖与容量的平衡点。是当前**4G/5G的核心频段**（如3.5GHz）。
  - **高频(>24GHz，毫米波)**：带宽极大，速率极高，但传播损耗大，覆盖小。用于**热点区域容量补充**。

- **非授权频段**：无需申请即可使用的公共频段，是Wi-Fi、蓝牙等技术的基石。
  - **2.4GHz ISM频段**：全球通用，但干扰严重。
  - **5GHz U-NII频段**：带宽更宽，干扰相对较少。

- **频谱共享技术**：为解决稀缺性问题，允许次要用户在**不影响主用户**的前提下使用授权频段。
  - **认知无线电**：设备可“感知”频谱使用情况并智能接入。
  - **授权共享接入(LSA)/公民宽带无线电服务(CBRS)**：通过数据库动态管理频谱接入权限。

## 1.6 标准

标准是确保全球设备**互联互通**和产业规模化的关键。制定过程通常是漫长而严谨的，涉及需求分析、技术提案、辩论、融合、详细规范和最终冻结。

- **主要标准化组织**：
  - **3GPP**：制定蜂窝通信技术标准（从3G到5G及未来）。其工作以“Release”（版本）的形式发布，如R15是5G第一版标准。
  - **IEEE**：制定局域网/个域网标准，最著名的是**IEEE 802.11**（Wi-Fi）和**IEEE 802.15**（如ZigBee）。
  - **ITU**：负责国际无线电频谱分配和全球电信标准的总体框架（如IMT-2020定义了5G的愿景和要求）。

- **标准化的价值**：
  - **经济性**：规模化生产大幅降低设备成本。
  - **互操作性**：确保不同厂商的设备可以无缝协同工作。
  - **创新加速**：为产业界提供了清晰的技术发展路线图。

## 总结

第一章勾勒了无线通信的宏大图景：从赫兹验证电磁波存在的实验室，到马可尼连通大西洋的壮举，再到今天每个人口袋里的智能手机和即将到来的智能万物互联时代。它指明了技术发展的**终极愿景**——一个无所不在、智能高效的连接世界，同时也毫不回避地列出了实现这一愿景必须克服的**根本性技术挑战**。最后，它梳理了构成当下无线世界的**生态系统**（蜂窝、Wi-Fi、蓝牙等），并点明了**频谱**这一生命线和**标准**这一游戏规则的重要性。本章为读者理解后续深入探讨具体技术的章节（如信道、调制、多天线等）提供了必要的背景和脉络。
