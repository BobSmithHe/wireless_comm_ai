# 第4章 无线信道的信道容量

## 4.1 引言：无线信道的极限在哪里？

**信道容量**是信息论的核心概念，由香农在1948年提出。它定义了在任意小错误概率下，通过信道可实现的**最大可靠传输速率**。对于无线信道，容量分析尤为重要，因为它揭示了衰落、干扰等无线特定因素如何影响理论性能极限，从而指导实际系统设计。

本章从简单的**加性高斯白噪声(AWGN)**信道开始，逐步引入**平坦衰落**和**频率选择性衰落**，探讨在不同信道状态信息(CSI)条件下，无线信道的容量表达式、计算方法和关键结论。

## 4.2 AWGN信道容量：香农公式的起点

### 4.2.1 连续时间AWGN信道
对于带宽为 \(B\) (Hz)的AWGN信道，其**香农容量公式**为：

\[
C = B \log_2\left(1 + \frac{P}{N_0 B}\right) \quad \text{(bits/s)}
\]

其中：
- \(P\)：平均接收功率（瓦特）
- \(N_0\)：噪声功率谱密度（W/Hz）
- \(\frac{P}{N_0 B}\) 是**信噪比(SNR)**

**重要推论**：
1. 容量随带宽 \(B\) 单调增加，但增加速度逐渐变缓。
2. 当带宽趋于无穷时，容量趋于有限值：
   \[
   C_\infty = \lim_{B \to \infty} C = \frac{P}{N_0} \log_2 e \approx 1.44 \frac{P}{N_0} \quad \text{(bits/s)}
   \]
3. 定义**频谱效率** \(\eta = C/B\) (bits/s/Hz)，则有：
   \[
   \eta = \log_2(1 + \text{SNR})
   \]

### 4.2.2 功率和带宽的权衡
在总功率 \(P\) 固定时，可通过增加带宽 \(B\) 来换取更低的信噪比，但容量存在上限。在带宽受限时，提高功率可线性增加容量（对数关系）；在功率受限时，增加带宽只能使容量对数增长。

## 4.3 平坦衰落信道的容量

平坦衰落信道可建模为：\(y = h x + n\)，其中 \(h\) 是复信道增益，\(n\) 是复高斯噪声。**瞬时信噪比**为 \(\gamma = |h|^2 P/(N_0 B)\)，是一个随机变量。

容量分析的关键在于**发送端对信道状态信息(CSI)的知晓程度**。考虑三种典型情况：

### 4.3.1 情况1：接收端已知CSI，发送端未知CSI
这是最常见的情况（如盲发射）。发送端以恒定功率 \(P\) 发送，接收端知道 \(h\) 并可相应调整解调。此时信道是**时变但已知在接收端**，容量定义为**各态历经容量**：

\[
C = E_h \left[ B \log_2(1 + |h|^2 \text{SNR}) \right]
\]

其中期望是对信道增益 \(h\) 的分布求取。对于瑞利衰落 (\(|h|^2\) 指数分布)，有：

\[
C = B \int_0^\infty \log_2(1 + \gamma \text{SNR}) \frac{1}{\overline{\gamma}} e^{-\gamma/\overline{\gamma}} d\gamma
\]

其中 \(\overline{\gamma} = E[|h|^2] P/(N_0 B)\) 是平均SNR。

**重要观察**：由于 \(f(x)=\log(1+x)\) 是凹函数，由詹森不等式，\(C \leq B \log_2(1 + \overline{\gamma})\)，即**衰落降低了容量**（相比AWGN信道在相同平均SNR下）。

### 4.3.2 情况2：发送端和接收端都已知CSI（瞬时值）
发送端可根据信道状态自适应调整功率。优化问题为：

\[
\max_{P(\gamma)} E_\gamma \left[ B \log_2\left(1 + \frac{P(\gamma) \gamma}{P_{\text{avg}}}\right) \right] \quad \text{s.t.} \quad E_\gamma[P(\gamma)] \leq P_{\text{avg}}
\]

解为著名的**注水功率分配**：

\[
\frac{P(\gamma)}{P_{\text{avg}}} = \begin{cases}
\frac{1}{\gamma_0} - \frac{1}{\gamma}, & \gamma \geq \gamma_0 \\
0, & \gamma < \gamma_0
\end{cases}
\]

其中 \(\gamma_0\) 是“注水线”，由功率约束 \(E[P(\gamma)/P_{\text{avg}}] = 1\) 确定。对应的容量为：

\[
C = B \int_{\gamma_0}^\infty \log_2\left(\frac{\gamma}{\gamma_0}\right) f_\gamma(\gamma) d\gamma
\]

**关键点**：
- 注水法将更多功率分配给信道好的时候（\(\gamma\) 大），信道差时（\(\gamma < \gamma_0\)）甚至不发射。
- 相比固定功率，自适应功率分配可获得**容量增益**，尤其在中低SNR时显著。

### 4.3.3 情况3：发送端已知CSI的统计特性，但不知道瞬时值
发送端知道 \(f_\gamma(\gamma)\)，但不知道当前时刻的 \(\gamma\)。此时仍可采用固定功率发射，容量同情况1。若允许功率随统计自适应（但非瞬时），优化问题与情况2类似，但 \(P(\gamma)\) 只能是基于统计的固定函数，而非瞬时函数。

## 4.4 中断容量：慢衰落信道的可靠通信

在**慢衰落信道**中，信道在一个码字传输期间基本不变。此时，各态历经容量概念不再适用，因为无法通过编码跨越所有信道状态。定义**中断容量**：

给定一个目标速率 \(R\)，**中断概率**为：

\[
P_{\text{out}}(R) = P[\log_2(1 + \gamma \text{SNR}) < R]
\]

即信道瞬时容量小于 \(R\) 的概率。反之，对于允许的中断概率 \(P_{\text{out}}\)，最大的可传输速率 \(R\) 称为**中断容量**。

**重要关系**：在瑞利衰落下，
\[
P_{\text{out}}(R) = 1 - e^{-(2^R-1)/\overline{\gamma}}
\]
可解得：
\[
C_{\text{out}} = \log_2\left(1 + \overline{\gamma} \ln\left(\frac{1}{1-P_{\text{out}}}\right)\right)
\]

中断容量描述了慢衰落信道中速率与可靠性的权衡：要获得更高可靠性（更低中断概率），必须降低速率。

## 4.5 频率选择性衰落信道的容量

频率选择性衰落信道的冲激响应是时延的函数，可看作**一组并行的独立子信道**（如通过OFDM技术实现）。设总带宽 \(B\) 划分为 \(N\) 个子带，每个子带带宽 \(\Delta f = B/N\)，近似为平坦衰落。

### 4.5.1 并行独立子信道模型
第 \(i\) 个子信道的增益为 \(H_i\)，噪声功率为 \(N_0 \Delta f\)。发送端总功率约束为 \(P_{\text{total}}\)。容量优化问题为：

\[
\max_{P_1,\dots,P_N} \sum_{i=1}^N \Delta f \log_2\left(1 + \frac{|H_i|^2 P_i}{N_0 \Delta f}\right) \quad \text{s.t.} \quad \sum_{i=1}^N P_i \leq P_{\text{total}}, \quad P_i \geq 0
\]

解仍然是**注水功率分配**，但在频域上进行：

\[
P_i = \left( \mu - \frac{N_0 \Delta f}{|H_i|^2} \right)^+
\]

其中 \(\mu\) 是注水线，使总功率等于 \(P_{\text{total}}\)。

### 4.5.2 连续频域注水
当子带数 \(N \to \infty\) 时，求和变为积分。设信道频率响应为 \(H(f)\)，则容量为：

\[
C = \max_{P(f): \int P(f) df = P_{\text{total}}} \int_0^B \log_2\left(1 + \frac{|H(f)|^2 P(f)}{N_0}\right) df
\]

最优功率分配为：

\[
P(f) = \left( \mu - \frac{N_0}{|H(f)|^2} \right)^+
\]

**关键结论**：频率选择性衰落信道的容量可通过**在频域上注水**达到，将更多功率分配给信道条件好的频带。

## 4.6 多天线（MIMO）信道的容量

MIMO信道是第10章的重点，这里先给出核心结论。设发送天线数 \(N_t\)，接收天线数 \(N_r\)，信道矩阵 \(\mathbf{H} \in \mathbb{C}^{N_r \times N_t}\)。接收信号为：

\[
\mathbf{y} = \mathbf{H} \mathbf{x} + \mathbf{n}
\]

### 4.6.1 发送端未知CSI，接收端已知CSI
假设发送总功率为 \(P\)，在各天线上平均分配。则各态历经容量为：

\[
C = E_{\mathbf{H}} \left[ \log_2 \det\left( \mathbf{I}_{N_r} + \frac{P}{N_t N_0} \mathbf{H} \mathbf{H}^H \right) \right]
\]

**在丰富散射环境中**，\(\mathbf{H}\) 的元素为独立复高斯随机变量。当 \(N_t, N_r\) 较大时，容量近似为：

\[
C \approx \min(N_t, N_r) \cdot B \log_2\left(1 + \frac{P}{N_0}\right)
\]

即容量随 \(\min(N_t, N_r)\) **线性增长**！这是MIMO的巨大优势：在不增加带宽和功率的前提下，通过多天线大幅提升容量。

### 4.6.2 发送端已知CSI
发送端可进行**奇异值分解(SVD)**：\(\mathbf{H} = \mathbf{U} \mathbf{\Sigma} \mathbf{V}^H\)，从而将MIMO信道转化为 \(\min(N_t, N_r)\) 个**并行独立子信道**。最优功率分配是在这些子信道上进行注水，容量为：

\[
C = \max_{\{P_i\}: \sum P_i = P} \sum_{i=1}^{\min(N_t, N_r)} \log_2\left(1 + \frac{P_i \sigma_i^2}{N_0}\right)
\]

其中 \(\sigma_i\) 是 \(\mathbf{H}\) 的奇异值。

## 4.7 容量与实用系统的差距

香农容量给出了理论极限，但实际系统因以下原因无法达到：

1. **有限码长**：香农极限要求无限长码字，实际码长有限。
2. **编码/调制复杂度**：达到容量需要先进的编码（如Turbo、LDPC码）和迭代接收，复杂度高。
3. **信道估计误差**：CSI不完美降低性能。
4. **干扰**：实际系统存在同频干扰，香农容量假设中通常只有加性白噪声。

**频谱效率的演进**：
- 2G (GSM): ~0.1-0.3 bits/s/Hz
- 3G (WCDMA): ~0.5-1 bits/s/Hz
- 4G (LTE): ~2-5 bits/s/Hz (高阶MIMO、OFDMA)
- 5G: 目标10-20 bits/s/Hz

尽管有差距，但容量理论指导了技术发展：从单天线到MIMO，从固定调制到自适应调制编码，从单载波到OFDM，都是向香农极限逼近的努力。

## 4.8 总结

本章系统分析了各类无线信道的容量：

1. **AWGN信道**：香农公式 \(C = B\log_2(1+\text{SNR})\) 是基准。

2. **平坦衰落信道**：
   - **各态历经容量**：适用于快衰落，是对所有信道状态的平均。
   - **中断容量**：适用于慢衰落，描述速率与可靠性的权衡。
   - **发送端CSI的价值**：通过注水功率分配可提升容量，尤其在中低SNR。

3. **频率选择性衰落信道**：通过频域注水，将功率集中到信道条件好的频带。

4. **MIMO信道**：容量随天线数线性增长，是大幅提升频谱效率的关键。

**核心启示**：
- 衰落降低容量（相比AWGN），但通过自适应技术（功率、速率调整）可部分弥补。
- 获取信道状态信息（尤其是发送端CSI）对提升容量至关重要。
- 多天线和频率选择性都是可利用的“维度”，通过注水等优化可接近容量极限。

容量分析不仅给出了性能上限，也指明了系统设计方向：追求更准确的信道估计、更智能的自适应算法、更高效的多天线技术。在后续章节中，将具体探讨如何通过调制、编码、分集等技术逼近这些极限。
